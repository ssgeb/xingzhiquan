# 星知圈项目提交流程

本次将项目整理为两个顶层目录后，提交到 GitHub 仓库：

- `backend/`：后端 Spring Boot 项目
- `frontend/`：前端 Vue/React 项目

## 操作流程

1. 新建合并仓库目录 `xingzhiquan`
2. 将后端内容复制到 `backend/`
3. 将前端内容复制到 `frontend/`
4. 清理不需要提交的内容
5. 初始化 Git 仓库
6. 添加远程仓库
7. 提交代码
8. 推送到 GitHub

## 使用的 Git 命令

```powershell
git init -b main
git remote add origin https://github.com/ssgeb/xingzhiquan.git
git add .
git commit -m "Initial commit: Xingzhiquan frontend and backend"
git push -u origin main
```
